# AlgoLab - Sorting Algorithms Performance Analysis

## Description
This project implements and analyzes sorting algorithms (Selection Sort and Insertion Sort) using C++. The performance is evaluated based on execution time and number of comparisons under different input conditions.

## Implemented Algorithms
- Selection Sort
- Insertion Sort

## Features
- Generates random and reverse input data
- Measures execution time using C++ chrono library
- Counts number of comparisons for each algorithm
- Tests multiple input sizes (50, 100, 150)

## Input Types
- Random data
- Reverse sorted data

## Metrics Used
- Execution Time (milliseconds)
- Number of Comparisons

## Technologies Used
- C++
- STL (vector, random)
- chrono library

## Purpose
The project demonstrates algorithm efficiency analysis and helps understand how input conditions affect sorting performance.
